# AlgoLog Frontend

AlgoLog frontend client built with Vue 3, TypeScript, and Vite.

## Tech Stack

- Vue 3
- TypeScript
- Vite
- Vue Router
- Pinia
- Vue Query
- Vitest
- ESLint

## Getting Started

```bash
npm install
npm run dev
```

## Available Scripts

```bash
npm run dev
npm run build
npm run lint
npm run test
npm run check
```

## Git Workflow

- Base branches: `main`, `develop`
- Feature branches: `feature/#issue-number-description`
- Pull requests target `develop`

## Templates

- Issue templates and PR template are synced from the backend repository conventions.
